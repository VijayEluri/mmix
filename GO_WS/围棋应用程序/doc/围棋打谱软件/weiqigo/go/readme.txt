          说明

本围棋打谱软件收集六千余盘中外棋局，运行环境为简体中文Windows95。
使用方法：
  1.  运行go.exe。
  2.  点菜单(文件-打开)，选择棋库。
  3.  点菜单(文件-选择)，选择棋局。
  4.  单击左键落一子，单击右键回一子。不击键则以2秒一手自动打谱。
  5.  使用试走功能，需要停止自动状态。
